# Crop Disease Detection using CNN and MobileNetV2

## Overview

This project detects plant diseases using deep learning.

## Models Used

* CNN (Custom Model)
* MobileNetV2 (Transfer Learning)

## Results

* CNN Accuracy: 87.6%
* MobileNetV2 Accuracy: 96.0%

## How to Run

Install dependencies:
pip install -r requirements.txt

Train models:
python train_cnn.py
python train.py

Run app:
python app.py
